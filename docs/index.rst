Welcome to *ccatkidlib*'s Documentation!
========================================

*ccatkidlib* is a Python library developed for readout and analysis of `CCAT Observatory's <https://www.ccatobservatory.org/>`_ kinetic inductance detectors (KIDs). 
CCAT uses `Xilinx ZCU111 Radio Frequency System on a Chip (RFSoC) <https://www.amd.com/en/products/adaptive-socs-and-fpgas/evaluation-boards/zcu111.html>`_ boards
for frequency-multiplexed readout of the detectors with the `primecam_readout <https://github.com/TheJabur/primecam_readout>`_ firmware and low-level software.
*ccatkidlib* interfaces with the low-level software to provide high-level functionality for detector calibration and readout.
